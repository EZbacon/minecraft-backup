
# 🧨My modpack in minecraft




  

## 📦 What’s This?

  

This is a full-blown Minecraft modpack stored on GitHub, using **Git LFS** to handle chunky files like `.jar` mods, zip configs, and more.

It's synced with my Modrinth launcher folder — mods, configs, and tweaks all backed up, no bullshit.

  

---

  

## 🚀 Quick Start (for players)

  

### 🧠 Requirements

- Minecraft installed (duh)

- Modrinth App or similar launcher

## 💾 One-Click Download

  

If you're lazy or just want the files:

  

- Click the green **Code** button

- Hit **Download ZIP**

- Extract it into your Modrinth folder

- Done.
